# 165. Compare Version Numbers

## 🧩 Problem Statement

Given two version strings `version1` and `version2`, compare them.

Each version string consists of revisions separated by dots `'.'`.

### Comparison Rules
- Compare revisions from left to right.
- Revisions are compared numerically.
- Leading zeros should be ignored.
- Missing revisions are treated as `0`.

Return:
- `1` if `version1 > version2`
- `-1` if `version1 < version2`
- `0` if both are equal
